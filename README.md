# MindSpace

"Small mindful moments, big mental wellness." 🌱

MindSpace is your daily mental wellness companion. It combines mindfulness, mood tracking, guided meditation, journaling, and AI-powered emotional support to help users improve their mental well-being every day.

## Core Features
*   **Mood Tracker:** Daily check-ins and insights to identify patterns.
*   **Guided Mindfulness Sessions:** Short and long exercises for stress relief and sleep.
*   **Smart Journal:** Gratitude journaling with AI-generated reflections.
*   **Mindful AI Coach:** Chat with an AI wellness companion for coping strategies.
*   **Community Support:** Anonymous discussion groups and daily positive challenges.
*   **Progress Dashboard:** Track mood trends, meditation minutes, and consistency.

## Tech Stack
*   Flutter for Cross-Platform App Development (iOS, Android, Web, Desktop)

## Setup
1. Ensure Flutter is installed on your system.
2. Run `flutter pub get` to fetch dependencies.
3. Run the app with `flutter run`.
